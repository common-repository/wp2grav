Export your Wordpress posts, pages and more into a markdown content structure for Grav

== Description ==

This plugin allows you to export your posts, pages, authors, tags and categories into a markdown file based content structure for getgrav.org

Currently following content types are supported

* pages
* posts
* authors

Following meta data is supported:

* publish data
* published status
* authors
* categories
* tags



== Installation ==

Upload the WP2Grav plugin to your site, and activate it! Yep, that's it!
